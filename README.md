# FNP Sales Dashboard using Excel

This project is an interactive sales dashboard created in Microsoft Excel using the FNP (Ferns and Petals) dataset. The dashboard was designed to analyze sales performance, customer behavior, product trends, and delivery insights in a simple and visually clear way.

## Project Objectives
The dashboard helps answer business questions such as:
- What is the overall revenue?
- Which products generate the highest revenue?
- How do sales change across different months?
- Which cities place the highest number of orders?
- Which occasions generate the most revenue?
- How does order quantity affect delivery time?

## Dashboard Features
- Revenue Analysis
- Monthly Sales Trends
- Top Products by Revenue
- Customer Spending Analysis
- Occasion-wise Revenue Comparison
- Top Cities by Orders
- Delivery Time Analysis
- Interactive Filters and Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization
